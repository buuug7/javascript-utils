# javascript utils

some javascript utils

## usage

```javascript
import utils from "@buuug7/javascript-utils";

const str = utils.parseTime(new Date());
```

or

```javascript
import { parseTime } from "@buuug7/javascript-utils";

const str = parseTime(new Date());
```
